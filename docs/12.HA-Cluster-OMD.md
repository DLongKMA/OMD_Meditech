### 1. Chuẩn bị

<a name="11" />
- **Yêu cầu hệ thống:**
	- Hệ điều hành CentOS 7
	- Mỗi server gắn thêm một ổ cứng 5GB
	- Mỗi server có 2 card mạng
	- Tắt Firewall và SELinux trên cả 2 server
	- Đồng nhất thời gian trên 2 server
